菲律宾

发表于 2017-08-18

​

印象

来到这里之前，对这个国家的印象是寥寥的几个关键词：香蕉，菲佣，黄岩岛, 杜特尔特… 来了后发现所有书籍游记攻略对这个国家的描述在她鲜活地铺开在面前后，对她之前印象几乎全部推翻。

安保人员绝对是菲律宾的一大亮点，看上去这个工种是全国统一考核，统一着装: 合身的白色上衣点缀金色盾标徽章，胸口绣着英文姓名工作单位，下身黑色长裤以及锃亮的皮鞋，腰间别了一圈东西，包括带十字标志的急救包，手铐，手枪，匕首，电棍，常用工具等等，加上他们一般都身材匀称，加上这身制服提神，看上去非常帅气。几乎每个稍微大点的商业单位都会配备这种带枪的保安，同时他又是店里的工作人员，在门口负责待人接物，开门提包之类的工作，应该也兼顾着治安的作用。相较而言，警察制服就逊色许多，交警的服饰就更不用提，他们看上去只是身上写着TRANFIC的普通工装。

经济

城市里高楼很少，不像中国随便一个二三线城市高楼大厦鳞次栉比。在马尼拉或者宿务能很明显地感觉到贫富差距，像首都市中心这样的地方，高楼在周围覆盖着大片破旧不堪的平房中拔地而起，即使只有二三十层也显得非常突兀。住在市中心酒店的二十几楼，早晨五点多就被四周平房里此起彼伏的鸡叫声，狗吠声，小孩哭闹声惊醒。街边大概不出三十米五十米就能看到流浪者拿着纸板躺在街边商铺屋檐底下，靠乞讨与捡拾破烂维生，其中不乏年轻人，甚至一家三口，带着小孩睡在地上。

几乎没见到过豪车，百分之九十都是日本车，车型也基本都是国内十万二十万的家用小汽车，剩下的百分之几是本地的一种叫吉普尼的公交车，据说是二战后美军留下的吉普改装加长的，大多造型浮夸，风格迥异。这个吉普尼组成的公共交通网非常发达，穿街走巷，随处能看到它们的身影，走在街上几乎一两分钟就能碰到一辆。它们跟国内公交一样有着固定的线路，招手即停，很是方便。乘客从后门钻进车里，车身异常低矮，进去后把钱从后窗递给司机，每到一站司机会把写着下一站名字的木板挂在后视镜处。

人们

每在这里多呆一天就对这里的人们好感多增一分，不是所有地方的穷山恶水都出刁民。这里的普通民众很穷，但是很淳朴。从另外一方面来说其实应该是聪明，不会为了短期个人利益坑你钱而失去长期大家的利益。从收入上来看，即使在马尼拉，一个普通出租车司机一天收入在600比索左右，合人民币八九十元，在偏僻郊远区域更是有突突（当地用两轮摩托改装带偏斗可以乘坐4-5人的交通工具）司机会为了一人50比索（人民币6块7）而等你一上午。
另我印象很深刻的一件事是，有一次晚饭后无事去沙滩散步，沙滩边有各种酒吧以及夜市。酒吧门口有性工作者拉生意，我当时坐在离她们不到十米的地方等人，看着她们一次又一次充满希望走向一个人又失望地回到酒吧门口，我等了半个多小时，她们也忙碌了半个多小时，没有一次成功的。我等人也没耐心，就走开去沙滩吹海风了。大概又过了十多二十分钟，回到等人的地方，其中一个性工作者坐在我刚才坐的地方，问我Sir你的手机呢？这个是不是你的？我才反应过来一摸兜确实丢了，估计是坐那从兜里滑出来了。那个人说1000rmb就给你，我当时大概懵了三四秒，结果她笑着说开玩笑的，你解锁证明是你的我就还给你，我半天没回过神。作为这个社会的最底层工作者她不会不知道如果捡走随便卖个价至少她应该好几天不需要在这里守候她的生意了吧，但是她没有。

脑洞

发表于 2017-08-06

这个世界所有事物都是由两部分构成: 物质与信息。拿这个世界上肉眼能看到的最复杂事物-人来举例: 肉体是物质，这些物质如何组织到一起是信息，人体每经过一个事件，外部感官系统将外界刺激通过神经元反馈到大脑，经过大脑加工形成记忆，这个过程中你这个个体并没有从外界吸收任何物质，只是有的分子经过反应后排列组合成新的分子，这种存储的实质追究到根源无非是组成你这个人的信息变了，使你这个个体变成了另一个带着新秩序的信息集合。这样看来，你的灵魂来自于组成你这个人的物质的所有信息。假设这世上有另外一个人，与你同样体重，同样分子含量，因为组成顺序不同，他拥有另外一个灵魂。相反，如果这些物质，精确到每个原子，夸克,甚至更微小级别的信息都一模一样，你们会连性格，灵魂都会是一模一样的。

由此可以推测出科学的边界应到此为止: 有一个可以获取这个物体所有信息的扫描装置，包括每个原子的位置，运动，状态等所有信息，再加上一个可以按这些顺序将原子精密排放的打印装置。

这是科学与上帝的界限。如果有这样一个实现，几乎所有事情都可以做到，任何人类可以通过这样一台装置变成神。器官捐献？不需要，我们反正是碳基生命，找个健康人扫描一下，打印一个。甚至只需要把你健康时的抓取的快照信息读取出来，用打印机修复一下就可以。缺钱？扫描一块钻石信息，拿块木炭重新排个序，想要几克拉就打几克拉。时空穿梭？扫描你的信息，通过无线电以光速把数据发送出去，接收方再用对应含量的物质把你组合出来。届时，所有物质存在几乎都不再有意义，你就是神。拥有这两个装置后你能做的事情几乎仅受限于你的想象，不过，这样打印重组出来的大米吃起来是不是有田间沐浴过时光的好吃？跟你结婚时的那个她被重组几次后还是那个她吗？那就是另外一个话题了。也许就跟现在有些产品明明机器制造质量要好过于手工打造，就是有人喜欢手工一样，明明整容后确实赏心悦目了，但是有人还是喜欢原装一样。到那时，两个物质一样，信息一样的个体，唯一的区别在于第三个观察者对它/他/她的情怀了。

还好，这个界限不是那么好打破，现实的禁锢在于，拿人体举例，一个人包含的所有信息(所有器官，细胞，记忆，细到每个原子的组成，排序，运动状态) 数量级至少是二的百次方级，那就意味着使用上这个宇宙已知能观测到的所有原子都无法表达一个人的信息。虽然听起来不可思议，但是就像十年前你没想过在网上下载一个文件就可以自己在家里用一台淘宝几千块买的3D打印机打出来一把真正能发射子弹的枪一样，这个脑洞理论上是可行的。

兜兜转转

发表于 2017-06-25

​
人生就是在很多while循环里兜兜转转，因果轮回，直到一个偶然或者必然的if条件分支跳出或者返回。不光人生，包括这个世界，甚至整个宇宙都倾向于稳定，循环，无序。没有任何条件变更的情况下，五年或者十年后你会执行到哪条语句哪行指令全部暗含在常量以及初始条件中，成为永远执行同一段代码的死循环。
假设这个世界是虚幻的，假设自己，甚至整个宇宙都是造物主写的一堆代码，我都不想假设，因为实在是太像了，几乎所有的事物包括时间发展都趋于一个方向，宇宙从奇点那一刻起，时间只往前走，连熵都只趋向于不变或增大，这多少有点让人绝望。回到过去是不可能的，若只如初见也是不可能的，重头再来更是扯淡。成不了bug，就加一个小小的i++, 虽然不知道哪天break，却总是充满希望。

知识与智慧

发表于 2017-05-21

​
为什么千百年来人类只是知识一直在积累增多，相对而言，智慧增加却极为有限，有些方面甚至说不好是古人更智慧还是现代人。在我看来，有一个重要的因素是知识可以很容易通过媒介传递，而智慧几乎只能自我产生而很难通过言传身教来灌输。智慧的产生可以通过在实践中获取经验、保持怀疑态度以及独立思考等方法或手段积累，个人认为最重要的途径在于对一切保持怀疑态度，哪怕是今天的真理。有怀疑才有积极的思考，才有去实践来获取智慧的动力。

Java 常用命令简介

发表于 2016-07-16

一、常用命令：

在 JDK 的 bin 目录下,包含了 java 命令及其他实用工具。

jps: 查看本机的 Java 中进程信息。

jstack: 打印线程的栈信息,制作线程 Dump。

jmap: 打印内存映射,制作堆 Dump。

jstat: 性能监控工具。

jhat: 内存分析工具。

jconsole: 简易的可视化控制台。

jvisualvm: 功能强大的控制台。

二、认识 Java Dump：

什么是 Java Dump？

Java 虚拟机的运行时快照。将 Java 虚拟机运行时的状态和信息保存到文件。

线程 Dump,包含所有线程的运行状态。纯文本格式。

堆 Dump,包含线程 Dump,幵包含所有堆对象的状态。二进制格式。

Java Dump 有什么用？

补足传统 Bug 分析手段的不足: 可在任何 Java 环境使用; 信息量充足。 针对非功能正确性的 Bug,主要为:多线程幵发、内存泄漏。

制作 Java Dump

使用 Java 虚拟机制作 Dump

指示虚拟机在发生内存不足错误时,自动生成堆 Dump

使用图形化工具制作 Dump

使用 JDK(1.6) 自带的工具: Java VisualVM。

使用命令行制作 Dump

jstack: 打印线程的栈信息,制作线程 Dump。
jmap: 打印内存映射,制作堆 Dump。

步骤：

  1. 检查虚拟机版本（java -version）
  2. 找出目标 Java 应用的进程 ID（jps）
  3. 使用 jstack 命令制作线程 Dump • Linux 环境下使用 kill 命令制作线程 Dump
  4. 使用 jmap 命令制作堆 Dump
